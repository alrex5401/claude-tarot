# 🎴 塔羅 Claude Skill — 2 分鐘安裝教學

嗨！這是一個塔羅牌解讀小工具，你自己抽牌，**Claude AI 幫你解讀**。

---

## 你需要什麼？

- 一個 **Claude 帳號**（claude.ai，免費版即可）
- 一副實體塔羅牌 🎴，或任何塔羅抽卡 App（Labyrinthos、Golden Thread 都可以）

沒有塔羅牌也沒關係，Google「免費塔羅抽卡」也能用。

---

## 方式一：免費版用戶（約 2 分鐘）

### 步驟 1：開新 Project

登入 [claude.ai](https://claude.ai)，在左側選單點 **「+ New Project」**，隨便取個名字（例如「塔羅小助手」）。

### 步驟 2：貼入角色設定

- 用文字編輯器（或系統內建 TextEdit）打開本資料夾裡的 **`skill/tarot/SKILL.md`**
- 把**全部內容**複製貼到 Project 右側的 **「Project instructions」** 欄位
- 點儲存

### 步驟 3：上傳牌義資料

- 在同一個 Project 頁面，找到 **「Project knowledge」**（或叫「Files」）
- 上傳本資料夾裡的 **`skill/tarot/cards.json`**

完成！現在進入這個 Project 開始對話就能用。

---

## 方式二：Pro / Max 訂閱用戶（約 1 分鐘）

如果你有 Claude Pro 或 Max 訂閱：

1. 點右上角頭像 → **Settings** → **Capabilities** → **Skills**
2. 點 **「Upload Skill」**
3. 把本資料夾的 `skill/tarot/` 整個壓縮成 ZIP 再上傳（記得包含 `SKILL.md` 和 `cards.json`）

之後在任何對話直接用 `/tarot ...` 呼叫。

## 怎麼用？

### 🎴 模式 A：單張問卜（有具體問題）

你**自己抽一張牌**，告訴 Claude 問題 + 牌名 + 正位/逆位：

```
問題：這週該不該跟老闆談薪資？
牌：愚者（正位）
```

Claude 會回覆：**牌象**、**解讀**、**建議** 三段（約 150-250 字）。

### 🎴 模式 B：三牌陣問卜（複雜議題）

抽三張，自訂位置標籤（過去-現在-未來 / 情況-行動-結果 都可以）：

```
問題：跟 A 的合作該不該繼續？
牌1（過去）：戰車（逆位）
牌2（現在）：聖杯五（正位）
牌3（未來）：世界（正位）
```

Claude 會分別解讀三張，再給整體脈絡（約 350-500 字）。

### 🎴 模式 C：每日能量（沒具體問題，想知道今天氛圍）

```
/tarot daily 牌：愚者（正位）
心情：對今天的會議緊張
```

Claude 會回覆：**今日能量**、**提醒**、**關鍵字** 三段（約 100-180 字）。心情可不填。

## 常見問題

**Q：我不知道正位還是逆位怎麼辦？**
A：實體牌抽出後，牌面朝你是正位，牌面顛倒（文字上下顛倒）是逆位。如果你習慣抽卡時不分正逆位，全部當正位就行。

**Q：Claude 會記得我之前抽的牌嗎？**
A：同一個對話內會記得，換新對話就重新開始。如果想累積長期日記，自己複製到備忘錄 / Notion。

**Q：牌名要打中文還是英文？**
A：都可以，`愚者` 或 `The Fool` 都接受。

**Q：可以問工作 / 愛情 / 健康以外的問題嗎？**
A：都可以，塔羅是反映工具，任何人生議題都能問。但 Claude **不會**給醫療、法律、財務的絕對建議，只給反思方向。

**Q：這會洩漏我的個人問題給別人嗎？**
A：你的對話只存在你自己的 Claude 帳號裡，Anthropic 會依 claude.ai 隱私政策處理。其他人看不到。

## 塔羅是什麼

塔羅不是預測工具，而是**反映工具**——透過隨機抽牌，讓你從不同角度看自己的處境。
解讀結果僅供參考思考，**不構成醫療、法律、財務建議**。
