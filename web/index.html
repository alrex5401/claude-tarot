常見問題

      

        

          為什麼要做這個工具？
          
叫 AI 解牌很容易，但 prompt 寫不好就只拿到罐頭答案。這頁幫你把 78 張牌的關鍵字、正逆位、牌陣位置打包成可用的 prompt——你抽，AI 讀。

        

        

          為什麼要 copy 去別的 LLM、不直接在這頁解？
          
不綁特定 AI、不付 API 錢、你的問題不經我手。你抽完拿到 prompt，想貼 ChatGPT 就貼 ChatGPT、想貼 Claude 就貼 Claude，自由切換。

        

        

          能不能離線用？
          
抽牌跟 prompt 生成都在你的瀏覽器、無網路也能跑。解讀要 LLM 才行，看你選的 ChatGPT/Claude/Gemini 是不是離線可用。
